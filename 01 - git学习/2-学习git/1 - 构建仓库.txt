构建本地仓库才能用git对我们的代码进行版本控制！！！

	1 - 在本地目录中初始化仓库
		01 - $ cd    <path>     # 打开（或新建）本地目录
		02 - 进入该目录，右键打开“git bath here”
		03 - 执行命令 $ git init
			-- 在目录结构中，看到包含有.git的子目录，说明版本库创建成功！
			- 现在我们的机器上有了： 一个真实项目的git仓库
			- 并且，从这个仓库中检测出了所有文件的 工作副本
			
		
	2 - 克隆现有仓库
		$ git clone <url>
			git支持多种 数据传输协议
				htpps://协议
				git:// 协议
				SSH 传输协议
